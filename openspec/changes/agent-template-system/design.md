## Context

AIHub定位为"像搭乐高一样构建AI Agent"的可视化平台。Agent模板系统是核心差异化功能，允许用户通过选择预置模板快速创建Agent，降低使用门槛。

## Goals / Non-Goals

**Goals:**
- 实现Agent模板的定义、存储、实例化
- 提供5+内置模板覆盖常见场景
- 支持模板继承便于扩展

**Non-Goals:**
- 模板市场交易功能(P2)
- 模板版本回滚(P2)
- 模板评分评价(P2)

## Decisions

### D1: 模板存储方式
- **选择**: 数据库存储 + JSON Schema
- **理由**: 与现有系统一致，支持CRUD管理，便于权限控制

### D2: 模板结构
- **选择**: 6组件配置(模板/模型/RAG/工具/技能/Prompt)
- **理由**: 与README中的"能力积木"理念一致

### D3: 模板继承
- **选择**: 单继承 + Mixin
- **理由**: 简化实现，满足大多数场景需求

## Risks / Trade-offs

- [风险] 模板版本管理复杂度 → [规避] MVP只做基础版本，延后复杂版本管理
- [风险] 模板市场数据量少 → [规避] 初期聚焦内置模板，社区模板作为长期目标
