# 教程

学习如何为不同场景创建 AI Agent。

## 场景 1: 企业知识问答

创建一个能够回答企业文档相关问题的 Agent。

### 步骤

1. **创建 Agent**
   - 模板: 通用
   - 模型: GPT-3.5 或 GPT-4

2. **配置 RAG 知识库**
   - 上传企业文档（PDF、Word、Markdown）
   - 系统自动向量化

3. **测试**
   - 提问: 「我们的退换货政策是什么？」
   - Agent 会从文档中检索相关信息回答

### 预计时间

2 分钟

---

## 场景 2: 智能客服

创建一个能够查询订单、用户信息的客服 Agent。

### 步骤

1. **创建 Agent**
   - 模板: 客服
   - 模型: GPT-4

2. **配置 MCP 工具**
   - 启用「订单查询」工具
   - 启用「用户信息查询」工具

3. **配置 Prompt**
   - 角色: 专业客服
   - 风格: 友好耐心

4. **测试**
   - 提问: 「我的订单到哪里了？」
   - Agent 会调用工具查询订单状态

### 预计时间

5 分钟

---

## 场景 3: 代码助手

创建一个能够帮助编写和审查代码的研发助手。

### 步骤

1. **创建 Agent**
   - 模板: 研发
   - 模型: Claude 或 GPT-4

2. **配置知识库**
   - 上传技术文档
   - 上传代码规范

3. **配置 MCP 工具**
   - Git 操作
   - 代码搜索

4. **测试**
   - 提问: 「帮我写一个用户登录的 API」
   - Agent 会生成符合规范的代码

### 预计时间

3 分钟

---

## 场景 4: 数据分析

创建一个能够分析数据、生成报表的助手。

### 步骤

1. **创建 Agent**
   - 模板: 分析
   - 模型: GPT-4

2. **配置 Skills**
   - 数据分析技能
   - Excel 操作技能

3. **配置 MCP 工具**
   - Excel 文件读取
   - 数据可视化

4. **测试**
   - 提问: 「分析上月的销售数据」
   - Agent 会分析并生成报告

### 预计时间

4 分钟
