# 系统架构

## 概述

AIHub 是一个 AI 能力编排平台，帮助用户快速构建和部署 AI Agent。

## 整体架构

```
                    ┌─────────────┐
                    │   前端      │
                    │  (Vue 3)   │
                    └──────┬──────┘
                           │
                           ▼
┌─────────────┐     ┌─────────────────┐     ┌─────────────┐
│   用户      │────▶│  API 网关       │────▶│  后端服务   │
└─────────────┘     │  (Higress)     │     └──────┬──────┘
                    └────────┬────────┘            │
                             │                      │
              ┌──────────────┼──────────────┐      │
              │              │              │      │
              ▼              ▼              ▼      ▼
       ┌───────────┐ ┌───────────┐ ┌───────────┐ ┌───────────┐
       │  平台服务  │ │ Agent服务  │ │ RAG服务   │ │ MCP服务   │
       │  (Java)  │ │ (Python)  │ │ (Python)  │ │ (Python)  │
       └───────────┘ └───────────┘ └───────────┘ └───────────┘
              │              │              │              │
              └──────────────┼──────────────┘              │
                             │                             │
                             ▼                             ▼
                    ┌─────────────────┐          ┌─────────────────┐
                    │   MySQL        │          │   Redis        │
                    └─────────────────┘          └─────────────────┘
```

## 服务说明

### 前端服务 (Vue 3)
- 管理界面
- Agent 创建/配置
- 对话界面

### 平台服务 (Java)
- 用户管理
- 角色权限
- 系统配置

### Agent 服务 (Python)
- LangChain Agent
- 对话管理
- 工具调用

### RAG 服务 (Python)
- 文档处理
- 向量检索
- 知识库管理

### MCP 服务 (Python)
- MCP Server
- 工具注册
- 外部系统集成

## 技术栈

| 组件 | 技术 |
|------|------|
| 前端 | Vue 3, Element Plus, Vite |
| API 网关 | Higress |
| 平台服务 | Java 17, Spring Boot 3.2 |
| Agent 服务 | Python 3.10+, FastAPI, LangChain |
| 数据库 | MySQL 8.0 |
| 缓存 | Redis |
| 向量库 | Chroma/Qdrant |

## 路由规则

| 路径 | 服务 |
|------|------|
| `/api/auth/*` | 平台服务 |
| `/api/users/*` | 平台服务 |
| `/api/agent/*` | Agent 服务 |
| `/api/rag/*` | RAG 服务 |
| `/api/mcp/*` | MCP 服务 |

## 数据流

1. 用户请求 → 前端
2. 前端 → API 网关
3. 网关 → 路由到对应服务
4. 服务处理 → 数据库/缓存
5. 响应返回 → 前端
