# v1.0 发布说明

**发布日期**: 2026-03-20

## 🎉 首个正式版本

AIHub v1.0 是我们的首个正式发布版本，专注于核心AI对话能力。

## ✨ 新增功能

### 模型管理
- 支持多厂商LLM配置（OpenAI/Claude/智谱/通义）
- 模型测试与连接验证
- 灵活的API配置

### 对话功能
- SSE流式响应，打字机效果
- 会话历史管理
- 上下文保持

### Prompt模板
- 10+内置模板
- 变量支持
- 自定义模板创建

### 系统管理
- 用户认证与权限
- 系统监控
- 操作日志

## 📦 包含组件

| 组件 | 版本 | 说明 |
|-----|------|------|
| aihub-web | 1.0.0 | Vue 3 前端 |
| aihub-api | 1.0.0 | Spring Boot 后端 |
| aihub-agent | 1.0.0 | Python AI服务 |

这是首个版本，无需升级。
