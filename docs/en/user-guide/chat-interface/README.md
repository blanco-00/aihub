# Chat Interface

> **Feature Status**: ✅ Available (v1.0+)
> **Last Updated**: 2026-03-20

## Overview

AIHub provides an intuitive chat interface with:

- 💬 Real-time streaming responses
- 📝 Session history management
- 🔍 History search
- 📤 Conversation export

## Basic Chat

1. Click "AI Chat" in the left menu
2. Select the model you want to use
3. Type your message and press Enter
4. View AI responses in real-time

## Streaming Response

AIHub uses SSE (Server-Sent Events) for streaming:

- ⚡ First token latency < 500ms
- 🎯 Typewriter effect
- 🔄 Real-time display

## Session Management

### View History

- Click "Session History" on the left
- Search by keyword for quick access
- Click to resume previous conversations

### Export Conversations

- Export to Markdown / JSON format
- Easy sharing and archiving
