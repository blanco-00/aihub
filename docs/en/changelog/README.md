## [1.0.0] - 2026-03-20

### Added
- ✨ **Model Management** - Multi-provider LLM configuration (OpenAI/Claude/Zhipu/Tongyi)
- ✨ **Basic Chat** - SSE streaming response, typewriter effect
- ✨ **Prompt Templates** - 10+ built-in templates (customer service, code assistant, data analysis, etc.)
- ✨ **Session Management** - History and context persistence
- ✨ **User Authentication** - JWT authentication and RBAC permission management
- ✨ **System Monitoring** - Server status and health checks

### Changed
- 📝 Refactored documentation structure with dual-track system
- 🎨 Optimized frontend UI for better user experience

### Fixed
- 🐛 Fixed session history display issue
- 🐛 Fixed model configuration save failure

## [0.1.0] - 2026-01-12

### Added
- 🎉 Initial release
