# v1.0 Release Notes

**Release Date**: 2026-03-20

## 🎉 First Official Release

AIHub v1.0 is our first official release, focusing on core AI conversation capabilities.

## ✨ New Features

### Model Management
- Multi-provider LLM configuration (OpenAI/Claude/Zhipu/Tongyi)
- Model testing and connection verification
- Flexible API configuration

### Chat Features
- SSE streaming response, typewriter effect
- Session history management
- Context persistence

### Prompt Templates
- 10+ built-in templates
- Variable support
- Custom template creation

### System Management
- User authentication and permissions
- System monitoring
- Operation logs

## 📦 Included Components

| Component | Version | Description |

| aihub-api | 1.0.0 | Spring Boot Backend |
| aihub-agent | 1.0.0 | Python AI Service |
