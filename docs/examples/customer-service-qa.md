# 客服问答示例

## 场景描述

创建一个能够回答常见客户问题的客服 Agent。

## 配置

### Agent 设置
- **模板**: 客服
- **模型**: GPT-4
- **语言**: 中文

### Prompt

```
你是一个专业的客服助手，需要：
1. 礼貌热情地回复客户
2. 准确回答产品相关问题
3. 对于无法回答的问题，引导客户联系人工客服
```

### RAG 知识库

上传以下文档：
- 产品手册
- 常见问题 FAQ
- 退换货政策

### MCP 工具

- 订单查询（可选）
- 工单创建（可选）

## 测试用例

| 输入 | 预期输出 |
|------|---------|
| 你们的退换货政策是什么？ | 详细介绍退换货政策 |
| 这款产品有什么特点？ | 介绍产品特点 |
| 我想投诉 | 引导联系人工客服 |

## 进阶配置

- 添加更多产品文档到知识库
- 集成 CRM 系统获取客户信息
