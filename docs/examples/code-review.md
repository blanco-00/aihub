# 代码审查示例

## 场景描述

创建一个能够自动审查代码的研发助手。

## 配置

### Agent 设置
- **模板**: 研发
- **模型**: Claude 3.5 Sonnet
- **语言**: 中英文

### Prompt

```
你是一个专业的代码审查专家，需要：
1. 检查代码风格是否符合规范
2. 识别潜在的安全问题
3. 建议代码优化方案
4. 评估代码可维护性

请使用中文回复，对每条问题给出严重程度评级。
```

### MCP 工具

- Git 操作
- 代码搜索
- IDE 集成（可选）

## 审查结果示例

| 严重程度 | 问题 | 建议 |
|---------|------|------|
| 🔴 高 | SQL 注入风险 | 使用参数化查询 |
| 🟡 中 | 缺少错误处理 | 添加 try-catch |
| 🟢 低 | 命名不规范 | 使用更描述性的变量名 |
