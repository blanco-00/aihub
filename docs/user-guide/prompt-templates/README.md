# Prompt模板

> **功能状态**: ✅ 已实现 (v1.0+)
> **最后更新**: 2026-03-20

## 概述

Prompt模板让你可以：

- 📋 复用常用Prompt
- 🎯 标准化输出格式
- 🔧 使用变量动态生成
- 📚 共享团队模板

## 内置模板

| 模板名称 | 用途 | 示例变量 |
|---------|------|---------|
| 通用助手 | 日常对话 | - |
| 代码助手 | 编程问题 | `{{language}}` |
| 数据分析 | 数据处理 | `{{data_type}}` |
| 文案撰写 | 营销文案 | `{{style}}` |
| 客服话术 | 客户服务 | `{{product}}` |
| 翻译助手 | 多语言翻译 | `{{source_lang}}`, `{{target_lang}}` |

## 快速使用

1. 进入「Prompt模板」
2. 选择一个模板
3. 填写变量（如有）
4. 开始对话

## 创建自定义模板

### 1. 基础结构

```
你是一个{{role}}，专注于{{domain}}。

用户问题：{{question}}

请以{{style}}的方式回答。
```

### 2. 变量语法

- `{{变量名}}` - 必填变量
- `[[可选内容]]` - 可选块
