# 对话界面

> **功能状态**: ✅ 已实现 (v1.0+)
> **最后更新**: 2026-03-20

## 概述

AIHub 提供直观的对话界面，支持：

- 💬 流式实时响应
- 📝 会话历史管理
- 🔍 历史搜索
- 📤 对话导出

## 基础对话

1. 点击左侧菜单「AI对话」
2. 选择要使用的模型
3. 输入消息，按回车发送
4. 实时查看AI回复

## 流式响应

AIHub 使用 SSE (Server-Sent Events) 技术实现流式响应：

- ⚡ 首字延迟 < 500ms
- 🎯 打字机效果
- 🔄 实时显示

## 会话管理

### 查看历史

- 点击左侧「会话历史」
- 搜索关键词快速定位
- 点击恢复历史对话

### 导出对话

- 支持导出为 Markdown / JSON 格式
- 方便分享和归档
