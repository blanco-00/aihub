# 模型管理

> **功能状态**: ✅ 已实现 (v1.0+)
> **最后更新**: 2026-03-20

## 概述

AIHub 支持多厂商大语言模型的统一管理，让你可以：

- 🔄 一键切换不同模型
- 📊 对比不同模型效果
- 💰 优化成本与性能
- 🔒 灵活配置API密钥

## 支持的模型厂商

| 厂商 | 模型示例 | 特点 |
|-----|---------|------|
| OpenAI | GPT-4, GPT-3.5 | 通用能力强，生态完善 |
| Anthropic | Claude 3, Claude 2 | 长文本，安全性高 |
| 智谱AI | GLM-4, GLM-3 | 中文能力强，性价比高 |
| 阿里云 | 通义千问 | 中文优化，企业服务 |

## 快速开始

### 1. 添加模型

进入「系统管理」→「模型配置」→ 点击「添加模型」

### 2. 配置参数

- **模型名称**: 显示名称
- **API地址**: 厂商API地址
- **API密钥**: 你的API Key
- **模型ID**: 如 `gpt-4`, `claude-3-opus`

### 3. 测试连接

点击「测试」按钮，验证配置是否正确。
